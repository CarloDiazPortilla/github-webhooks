# GitHub Webhooks to Discord Notifier
A Node.js application that listens to GitHub webhooks (Stars and Issues events) and sends real-time notifications to Discord using a webhook.

## Requirements
- Node.js
- Discord
- GitHub repository
- Ngrok
- Ruby (optional to create a secure secret token)

## Running in development mode
1. Run the following command to install project dependencies:
```bash
npm install
```
2. Make a copy of `.env.example` and rename the file with `.env`.
3. To set the `SECRET_TOKEN` environment variable run the following command and save the output:
```bash
ruby -rsecurerandom -e 'puts SecureRandom.hex(20)' 
# example: 760ea6339dab54964d8cdce651526acb8b048aa6
```
4. Run the server with the following command in port 3000:
```bash
npm run dev
```
5. Make a tunnel with `ngrok` with the command and save the public url.
```bash
ngrok http 3000
```
6. Go to a GitHub repository that you want to receive notifications about.
7. Go to `Settings` > `Webhoooks` and click on `Add webhook`.
8. Set the Webhook with the following configuration:
  - `Payload URL`: Url generated by `ngrok` by tunneling port 3000 in step 5.
  - `Content type`: application/json
  - `Secret`: Secure random secret generated in step 3.
  - `SSL verification`: Enable SSL verification
  - `Which events would you like to trigger this webhook?`: Let me select individual events (select `Stars` and `Issues`)

> [!WARNING]
> This project only supports `Stars` and `Issues` GitHub events, other events will be ignored and logged as "Unknown event".

9. To set the `DISCORD_WEBHOOK_URL` create a server or use an existing one, then go to `Server setting` > `Integrations` > `Webhooks` > `New webhook`.
10. Click on the Webhook and click on the `Copy webhook URL` and set it in the `DISCORD_WEBHOOK_URL` variable environment.

> [!TIP]
> Now every event related to issues and stars in the repository will be logged in the discord server.
